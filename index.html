Before You Buy

Frequently asked

How do I choose between the 1/2-cup and 3/4-cup silhouettes?
The 25mm41-55 series use 1/2-cup coverage for a lower front profile. The 25mm56-70 series use 3/4-cup coverage for more front coverage. All six series use polyamide fabric and underwire construction.
What is the fabric, and does it run hot?
The bras are made with polyamide fabric. The finish changes by series: some use fabric cups with lace wings, some use lace cups with lace wings, and some use fabric cups with fabric wings.
The set includes five colors — which ones?
Every 5-pack ships with Midnight Navy, Powder Rose, Lavender, Sage Mist and Noir. Each bra is individually wrapped inside the box so they arrive clean and ready to wear.
How do I know which size to order?
Start with your current US bra size and compare it with the available Amazon size options. If you are between sizes or unsure, check the Amazon listing details and return policy before ordering.
How do I care for polyamide bras?
Delicate wash, cold, inside a mesh laundry bag. Air dry and avoid tumble drying. Clasp the hooks before washing to help protect the lace and fabric wings.
What is your return policy?
We sell exclusively through Amazon — which means Amazon's returns policy applies. You have 30 days from receipt to return the set for a full refund, and Prime members receive the usual free return shipping.
